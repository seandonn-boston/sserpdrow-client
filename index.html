Q&A

What is Sserpdrow?

Sserpdrow is a recreation of the basic principles of Wordpress. Sserpdrow allows users to create, read, edit, and delete blog posts or web pages.


How does Sserpdrow work?

By design, every visitor of this website, signed-in or otherwise, can view all the blog posts and web pages published by any user. New users can sign up (Join) then sign in (Login) using real or fake credentials, or you can use these provided example credentials to sign in:

u: sserpdrow@example.com
p: sserpdrow

When a user successfully signs in, that user can change their password (please do not edit the example account password) or sign out. A signed-in user will also see the "Create Post" section where they can create new blog posts or web pages by specifying a post type, title, and content, and saving the post by clicking the "submit" button. After a user has created their own post, it will appear as the last card in the "Blog Posts and Web Pages" section. If the current user created any of these blog posts or web pages, that user also has the ability to edit or delete that entry.

Sserpdrow often prompts the user with success and error messages as well which informs the user if something has or has not happened, such as failing to sign up because the username was already taken. These user messages appear above the "Blog Posts and Web Pages" section.


What's next for Sserpdrow?

Sserpdrow is under construction and several key features are still being developed, in particular a style guide, a list of users to filter all posts by, and the ability to have two or more co-owners on each post. Ultimately what is available at this time is the minimum viable product, which was contrusted to meet requirements for a scholastic group assignment - details of that assignment can be found here.
